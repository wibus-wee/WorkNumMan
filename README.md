### How to import students list?

**_Students List Format:_**

```
12345 // Room ID
StudentID,Name,Phone
StudentID,Name,Phone
...
```

### Format Keywords

- `<id>`: Student ID
- `<name>`: Student Name
- `<any>`: Any length of any string
- `<order>`: Order of Student
- `<order_CN>`: Order of Student in Chinese
